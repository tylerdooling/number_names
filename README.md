Number Names
======
-----

Description
-------

A simple gem that converts numbers in ruby to their respective human readable names.

Example
----

    require 'number_names'
    
    1.to_name => "one"
    301.to_name => "three hundred and one"
    3301.to_name => "three thousand three hundred and one"
